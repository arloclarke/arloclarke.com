# arloclarke.com

Static personal site for [arloclarke.com](https://arloclarke.com).

## Local preview

```bash
python3 -m http.server 8080
open http://127.0.0.1:8080
```

## GitHub Pages

This repo is prepared for GitHub Pages from the repository root:

- `index.html` is the site entrypoint.
- `CNAME` contains `arloclarke.com` for the custom domain.
- `.nojekyll` disables Jekyll processing.

When ready, create/push the GitHub repo and enable Pages from the `main` branch root.
